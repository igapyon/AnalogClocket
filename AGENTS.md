# AGENTS.md

このリポジトリは、単一 HTML で動作するアナログ時計 + ミニフリップ日付の Static Web App です。

## 目的と方針
- Static Web App として配布しやすい構成を維持する
- 見た目の良さと軽快な動作を重視する
- 更新は秒境界に合わせ、不要な再描画を避ける
- 静的要素は初期描画とリサイズ時のみ再描画する
- 日付などの情報は変化時のみ更新する

## 中心ファイル
- 主要な作業対象: `index.html`

## 実行方法
- `index.html` をブラウザで直接開く（ビルド不要）
- GitHub Pages で公開する場合はリポジトリのルートに配置する

## 外部ライブラリ
- SVG.js（CDN）

## 調整ポイント（代表）
- 日付の大きさ: `DATE_SCALE`
- 日付の位置: `r`（中心からの距離）
- 曜日位置: 日付ブロック内の曜日ラベル位置
